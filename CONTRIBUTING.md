# Contributing to Media-Careers-India

Thanks for your interest in contributing!

Guidelines:
- Please open an issue to discuss major additions or changes before submitting PRs.
- Keep contributions focused and well-documented.
- Add links with a short description and source.
- Use clear commit messages.

How to contribute:
1. Fork the repository.
2. Create a branch for your fix/feature: git checkout -b my-branch
3. Make your changes, commit, and push.
4. Open a pull request describing your changes and why they help.

If you want help getting started, open an issue and tag it "good first issue".
